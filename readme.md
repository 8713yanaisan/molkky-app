# モルック練習記録アプリ

## 🧩 概要
モルックの練習結果を記録・分析するWebアプリです。  
命中率やブレイク精度を可視化し、上達をサポートします。

---

## 🚀 主な機能

### 🎯 練習記録
- 距離（3.5m〜12m / 0.5m刻み）を選択
- 3×3ヒートマップで入力
  - ○：命中
  - 矢印：外れ方向
- 投球履歴表示
- 命中率（%）表示
- データ保存

---

### 💥 ブレイク練習
- 0〜12点のボタン入力
- 投球履歴表示
- 平均点の自動計算
- データ保存

---

### 📊 分析機能
- ユーザー別命中率
- 距離別命中率
- ブレイク平均点

---

### 👥 ユーザー機能
- ニックネーム設定
- ユーザーID自動生成
- 複数人データ管理

---

### 🔄 データ管理
- JSON形式でエクスポート
- JSONをインポートして共有
- データリセット機能

---

## 📱 画面構成

### 🏠 メニュー
- 練習
- ブレイク
- 名前設定
- 分析
- エクスポート
- インポート
- データリセット

---

### 🏃 練習フロー
1. 距離選択
2. スタート
3. ヒートマップ入力
4. 保存

---

### 💥 ブレイクフロー
1. 点数入力
2. 履歴確認
3. 保存

---

### 📊 分析
- 命中率
- 距離別分析
- ブレイク平均

---
